# THE COMMANDER FORGE · CMD

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **ARENA**.

THE COMMANDER FORGE (an ARENA tool sphere, not a set-browser) — pick a COMMANDER (Scryfall autocomplete), a STYLE (aggro/midrange/control/combo/stax/big-mana), a STRATEGY engine (tokens/+1+1-counters/aristocrats/spellslinger/reanimator/lifegain/artifacts/landfall/voltron/mill), and a FAVORITE CATEGORY — and it forges an algorithmic deck SKELETON + the best COMBO for your colors. ⚑ Source = EVERY Magic card ever printed, pulled LIVE from the Scryfall API (no fabrication; that database IS 'all cards ever made'), filtered to the commander's COLOR IDENTITY (so every pick is Commander-legal in your deck) and ranked by `order=edhrec` — Scryfall's real play-popularity rank, i.e. what the most decks actually run (verified: ramp→Sol Ring, removal→Swords, board-wipe→Cyclonic Rift). Builds 8 slots (ramp/draw/removal/wipes/strategy/style/favorite/lands) to a ~99 target, with a copy-able decklist seed, and matches a CURATED set of real famous combos (Thassa's Oracle+Consultation, Mikaeus+Triskelion, Kiki+Resto, Heliod+Ballista, Isochron+Dramatic Reversal, Basalt+Rings, …) whose color requirement fits your commander + strategy. ⚑ HONEST: 'best' = a transparent most-played + on-theme HEURISTIC, not a solver/guarantee, and not full EDHREC synergy; a strong starting skeleton to tune. Card names/prices/images are Scryfall facts (CDN hot-link, sanctioned non-commercial); fan tool, Magic © WotC. VERIFIED live (Atraxa BGUW → 79 cards, Sol Ring #1, Mikaeus+Triskelion combo, 0 errors).
